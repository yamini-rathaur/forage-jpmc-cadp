# forage-jpmc-cadp
This repository contains my solution to the [JPMorgan Chase & Co. Corporate Analyst Development Program](https://www.theforage.com/virtual-internships/4dy5SP43KMPppKz2i?ref=E9ar33oZo8bP5Ytks) on Forage(formerly InsideSherpa).

## Introduction
The Corporate Analyst Development Program (CADP) JPMorgan Chase & Co. (JPMC) gives participants the unique opportunity to develop technical, analytical, ad project management skills in different roles. 

Through the five modules in this virtual experience, one will have the opportunity to practice key skills by using important tools necessary for sucess in the three different CADP rotations, and in any role at JPMC. 

## Task1: Data Analysis and Simple Visualization in Excel
**Skills:** Excel, Data analysis, Written communication, Excel charts

## Task2: Data Visualization Dashboard
**Skills:** Tableau, Data visualization, Dashboards

## Task3: Process Mapping and Improvement
**Skills:** Process mapping, Process improvement, Written communication

## Task4: Storytelling with PowerPoint
**Skills:** Written communication, Storytelling

## Task5: Project Update Video
**Skills:** Verbal Communication, Project Management
